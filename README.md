本项目记录笔者积累的Java反序列化利用链的代码演示与相关文字描述。

根据每一个链，都单独建立了一个项目，在复现时可以单拎一个目录到IDEA里。

每一个链的文字描述都在对应项目下的README.md文件中。

Gadget清单：
> 笔者按照如下顺序依次完成项目的构建与文字描述，因此在写文时存在思路上的连贯性，并不构成学习顺序上的暗示。

* ✅ CommonsCollections1
* ✅ JNDIForHighVersionJDKWithLocalFactory
* ✅ C3P0

* ✅ CommonsCollections6

* 🚩 CommonsCollections2

* 🚩 CommonsBeanutils1
* 🚩 Groovy1
* 🚩 FileUpload1
